# Dashboard-Movimientos
Dentro de las medianas empresas de distribución la capacidad de almacenamiento puede ser un punto critico para la custodia de sus productos y en algunos casos se hace necesario la contratación de un servicio de almacenamiento. Contar con aplicaciones para la gestion de los movimientos entre los almacenes permite tener un proceso mas optimizado y agil para la toma de decision y soporte de las metas de la empresa en terminos de disponibiloidad y venta.
